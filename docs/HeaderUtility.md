# HeaderUtility

`HeaderUtility` is a static utility class designed to simplify common operations on HTTP headers within the `dotnet-api-gateway` project. It provides methods for retrieving, setting, adding, removing, and inspecting headers, as well as parsing authentication-related headers and copying headers between collections. This utility abstracts low-level header manipulation, ensuring consistency and reducing boilerplate code when working with headers in HTTP requests and responses.
