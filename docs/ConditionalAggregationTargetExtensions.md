# ConditionalAggregationTargetExtensions

Extension methods for `ConditionalAggregationTarget` that provide fluent configuration of conditional aggregation targets in API Gateway routing scenarios. These methods enable dynamic modification of aggregation targets based on runtime conditions such as HTTP method, headers, request body content, or timeout settings.

## API

### `Clone`
Creates a deep copy of the current `ConditionalAggregationTarget` instance. Useful when building multiple similar aggregation targets with minor variations.
